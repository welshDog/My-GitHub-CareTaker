## üì¶ Installation & Usage

### 1. Clone this repo

```bash

cd My-GitHub-CareTaker
```

### 2. Start the web UI (JS server)

```bash
cd caretaker-js/server
npm install
npm start
```

### 3. (Optional) Run the Python CLI

```bash
cd ../..
pip install -r requirements.txt
python caretaker_cli.py --username welshDog
```
